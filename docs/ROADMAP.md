# Roadmap

## Phase 1 — MVP (2–3 weeks)

- DOCX upload
- CSV/XLSX upload
- 2-column mapping UI
- Single DOCX generation
- Batch DOCX + ZIP
- Mapping export/import

Result: usable internal tool.

## Phase 2 — UX & Format Upgrade

- DOCX preview
- PDF export
- Output options (DOCX/PDF)
- Merge into one document
- Batch progress UI

## Phase 3 — Backend Introduction

- Backend for heavy batch jobs
- Persistent templates & mappings
- Job queue
- File storage
- API-based generation

## Phase 4 — Intelligence & Automation

- AI-assisted field mapping
- OCR input
- Legal templates validation
- Auto-filled metadata
- Integrations with external systems

## Phase 5 — SaaS / Enterprise

- Accounts
- Multi-tenancy
- Audit logs
- Access control
- SLA & monitoring
